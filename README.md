### Install

**As Zip**
```
wget -c https://github.com/undercodeutilities/Accesslist/archive/main.zip -O Accesslist.zip \
  && unzip Access.zip \
  && rm -f Access.zip
```


**Git**
```
git clone https://github.com/undercodeutilities/Accesslist.git
```
